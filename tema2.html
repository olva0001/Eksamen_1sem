Tema 2 - Grundlæggende Web

1. Semesters tema 2 introducerede os til programmeringssprogende HTML og CSS, herunder hvordan vi bruger dem til at bygge responsive hjemmesider. Temaet introducerede os også til de faglige begreber og værktøjer som vi skal bruge på resten af studiet, herunder Visual Studio Code til kodning, og figma til digitale prototyper.

Studiestartsprøven

På tema 2 skulle vi ud fra den viden vi havde opnået, fremstille et site ud fra udleverede wireframes, layoutdiagrammer og billedemateriale. Dette site skulle omhandle computere, inklusive viden om vores egen computers specifikationer. Sitet skulle afleveres som en del af studiestartsprøven, som var en forudsætning for fortsat indskrivning på studiet.

Sådan løste jeg opgaven

Til at begynde med, skulle siden kodes mobile first. Dette blev til afleveringen 02.01.01 Mobilsite. Dernæst blev vi introduceret til responsiv kodning, og vores næste skridt var at kode siden om til et Website, der fungerede optimalt på større skærme.

Du kan se min allerførste hjemmeside herunder!
